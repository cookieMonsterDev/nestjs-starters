# GraphQL api with auth and user flow using NestJS, Prisma 

## Environment variables

```bash
DATABASE_URL="database url"
AUTH_TOKEN_SECRET="auth token secret"
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
